# Contributing to Mera Dost

Thank you for your interest in contributing to Mera Dost!

**Project Maintainer:** Muhammad Sharjeel

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone <your-fork-url>`
3. Install dependencies: `npm run install:all`
4. Create a branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Run tests: `npm test`
7. Lint code: `npm run lint`
8. Commit with clear messages
9. Push and create a Pull Request

## Code Style

- Use TypeScript strict mode
- Follow ESLint and Prettier configurations
- Write meaningful commit messages
- Add JSDoc comments for public APIs
- Keep functions small and focused

## Testing

- Write tests for new features
- Maintain 60%+ test coverage
- Run E2E tests before submitting PR

## Pull Request Guidelines

- Clear description of changes
- Link to related issues
- Ensure CI passes
- Request review from maintainers

## Questions?

Open an issue for questions or discussions.
